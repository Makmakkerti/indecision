# Indecision app - let computer to choose for you!

Add options and let computer to choose randomly what you need to do!

Made for fun. Never do it in real life! ;)

## Tecnologies:
* React
* ReactDOM
* Webpack
* Babel
* Heroku

# Installation process
## Install
npm install

## Build and watch
npm run build

## Live-server to run locally
npm run serve
